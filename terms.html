O aplicativo Ric Lives utiliza os acessos da conta Facebook para automatizar a transmissão de lives para páginas que o usuário tenha acesso.

  São coletadas as seguintes informações dos usuários logados:

  - E-mail do usuário;
  - Lista de páginas que o usuário tem acesso como adminstrador;
  - Tokens de acesso às páginas do usuário.

  Como essas informações são utilizadas pelo sistema, via APIs:

  - Listar páginas que o usuário tem acesso como adminstrador;
  - Iniciar e encerrar lives para essas páginas;
  - Criar registros de logs das transmissões.

  Nenhum outro uso é feito com os acessos e informações colatadas pelo sistema.
